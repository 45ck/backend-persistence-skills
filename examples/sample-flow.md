# Sample Flow

## Prompt
"We need to design the backend for a course extension request system with a relational database, REST API, approval workflow, and auditable history. We need clean entities, migrations, and a consistent error contract."

## Suggested flow
1. entity-model-designer
2. schema-normalizer
3. crud-surface-reviewer
4. repository-layer-designer
5. jdbc-vs-jpa-selector
6. orm-mapping-reviewer
7. transaction-boundary-checker
8. query-complexity-reviewer
9. data-migration-planner
10. persistence-test-planner
11. api-contract-writer
12. api-error-model-designer

## Expected artifacts
- entity and relationship model
- normalized schema review
- repository/DAO plan
- transaction boundary notes
- migration plan and risks
- API contract and error model
